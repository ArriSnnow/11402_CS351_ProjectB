# Project B - CSV Mini Database and Query Engine

This project implements a compact CSV-based database and query engine in C++.

Key features:
- Load and parse CSV files with support for headers and typed values.
- Store data in an in-memory structure for efficient access.
- Execute basic SQL-like queries, including `SELECT`, `WHERE`, `ORDER BY`, and `LIMIT`.
- Support comparisons, logical expressions, and simple aggregation operations.
- Provide a small command-line interface for querying CSV datasets.

The goal is to build a lightweight database engine that can manage CSV data and process queries without relying on an external DBMS.
